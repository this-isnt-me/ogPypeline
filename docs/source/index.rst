ogPypeline documentation
===================================

**ogPypeline** is a Python library for the oil and gas industry. Initial release is focused on 64 useful unit conversions across multiple disciplines.

Check out the :doc:`installation` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
